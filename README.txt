== I. What is Ophal? ==

Ophal is a highly scalable web platform, easy to maintain, learn, extend
and open to improvements.

The reason for its name is a secret.


== II. Roadmap ==

Plans vary eventually, get up to date information here: http://ophal.org/roadmap


== III. Dependencies
Ophal has the following dependencies:
- Seawolf (http://github.com/ophal/seawolf)
- LuaSocket
- LPEG
- LuaFilesystem
- LuaDBI
- luuid
- dkjson
- LuaCrypto (only if user module is enabled)
